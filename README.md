## Uebersicht

Dieses Projekt ist eine Next.js-App fuer die ToolCompany Portfolio Plattform. Alle Oberflaechen, APIs und E-Mails sind auf Deutsch ausgelegt.

## Entwicklung starten

```bash
npm install
npm run dev
# oder
yarn dev
```

Die App laeuft unter `http://localhost:3000`.

## Wichtige Hinweise

- Metadata und Oberflaechen sind komplett deutsch; es gibt keinen Sprachumschalter.
- Seeds und Schema nutzen deutsche Werte (z.B. Statuswerte wie `Aktiv`).
- Passwort-Reset-E-Mails sowie API-Fehlermeldungen sind ebenfalls auf Deutsch gehalten.
